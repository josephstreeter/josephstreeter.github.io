## A

### Agent

An autonomous AI system that can perceive, reason, and act.

### Algorithm

A step-by-step procedure for solving a problem.

### Artificial Intelligence (AI)

Computer systems capable of performing tasks that typically require human intelligence.

### Attention Mechanism

A technique allowing models to focus on relevant parts of input.

## B

### Backpropagation

Algorithm for training neural networks by adjusting weights.

### Bias

Systematic error in AI predictions or unfairness in outcomes.

### BERT

Bidirectional Encoder Representations from Transformers.

## C

### ChatGPT

OpenAI's conversational AI model.

### Classification

Categorizing input into predefined classes.

### Clustering

Grouping similar data points together.

### Computer Vision

AI field focused on image and video understanding.

### Contextual Learning

Learning from context provided in prompts.

## D

### Deep Learning

Machine learning using multi-layer neural networks.

### Diffusion Models

Generative models that create images by reversing noise addition.

## E

### Embedding

Numerical representation of data in vector space.

### Encoder-Decoder

Architecture with separate encoding and decoding components.

## F

### Few-Shot Learning

Learning from a small number of examples.

### Fine-Tuning

Adapting pre-trained models to specific tasks.

## G

### Generative AI

AI systems that create new content.

### GPT

Generative Pre-trained Transformer.

## H

### Hallucination

AI generating false or nonsensical information.

### Hyperparameter

Configuration setting for model training.

## I

### Inference

Making predictions using a trained model.

### Instruction Tuning

Training models to follow instructions.

## L

### Large Language Model (LLM)

AI model trained on vast amounts of text data.

### LoRA

Low-Rank Adaptation for efficient fine-tuning.

## M

### Machine Learning

AI systems that learn from data.

### Model

Trained AI system that makes predictions or generates outputs.

### Multimodal

Processing multiple types of input (text, images, audio).

## N

### Natural Language Processing (NLP)

AI field focused on understanding and generating human language.

### Neural Network

Computing system inspired by biological neural networks.

## O

### Overfitting

Model performing well on training data but poorly on new data.

## P

### Parameter

Learned weights in a neural network.

### Prompt

Input instruction or query given to an AI model.

### Prompt Engineering

Crafting effective prompts for AI systems.

## R

### RAG

Retrieval-Augmented Generation.

### Reinforcement Learning

Learning through trial and error with rewards.

### RLHF

Reinforcement Learning from Human Feedback.

## S

### Supervised Learning

Learning from labeled training data.

### Self-Attention

Mechanism allowing models to weigh importance of different parts of input.

## T

### Temperature

Parameter controlling randomness in model outputs.

### Token

Unit of text processed by language models.

### Training

Process of teaching an AI model from data.

### Transfer Learning

Applying knowledge from one task to another.

### Transformer

Neural network architecture using attention mechanisms.

## U

### Unsupervised Learning

Learning from unlabeled data.

## V

### Vector Database

Database for storing and retrieving embeddings.

## Z

### Zero-Shot Learning

Performing tasks without specific training examples.
