# Photoshop Essentials

Explore advanced editing techniques using Adobe Photoshop for complex retouching, compositing, and creative image manipulation. This guide covers essential tools and professional workflows.

## Getting Started with Photoshop

### Photoshop vs Lightroom

#### When to Use Photoshop

- **Complex retouching**: Skin blemish removal, object removal
- **Compositing**: Combining multiple images
- **Precise selections**: Advanced masking techniques
- **Text and graphics**: Typography and design elements
- **Advanced effects**: Filters and creative processing
- **Fine art creation**: Digital painting and artistic effects

#### When to Use Lightroom

- **RAW processing**: Initial image development
- **Batch editing**: Processing multiple similar images
- **Color grading**: Global color adjustments
- **Library management**: Photo organization and cataloging
- **Basic retouching**: Simple spot removal and adjustments

### Photoshop Interface Overview

#### Workspace Layout

- **Tools panel**: Essential editing tools on left
- **Options bar**: Context-sensitive tool settings
- **Panels**: Layers, History, Properties on right
- **Canvas area**: Main image editing space
- **Menu bar**: Access to all Photoshop functions

#### Essential Panels

- **Layers**: Layer management and organization
- **History**: Undo/redo state management
- **Properties**: Adjustment layer controls
- **Channels**: Color channel information
- **Paths**: Vector path creation and editing

## Fundamental Concepts

### Layers System

#### Layer Types

- **Background layer**: Base image layer
- **Image layers**: Pixel-based content
- **Adjustment layers**: Non-destructive color/tone adjustments
- **Text layers**: Editable typography
- **Shape layers**: Vector-based graphics
- **Smart Objects**: Referenced layer content

#### Layer Properties

- **Opacity**: Layer transparency (0-100%)
- **Blend modes**: How layers interact with layers below
- **Layer masks**: Control layer visibility
- **Layer styles**: Effects like drop shadow, stroke
- **Fill**: Layer content opacity (separate from opacity)

### Blend Modes

#### Normal Blend Modes

- **Normal**: Standard layer blending
- **Dissolve**: Random pixel transparency
- **Darker Color**: Compares layers, shows darker
- **Lighter Color**: Compares layers, shows lighter

#### Darkening Blend Modes

- **Multiply**: Darkens by multiplying colors
- **Color Burn**: Increases contrast, darkens
- **Linear Burn**: Decreases brightness
- **Darker Color**: Shows darker of two colors

#### Lightening Blend Modes

- **Screen**: Lightens by inverting and multiplying
- **Color Dodge**: Brightens based on blend color
- **Linear Dodge (Add)**: Brightens by adding values
- **Lighter Color**: Shows lighter of two colors

#### Contrast Blend Modes

- **Overlay**: Combines Multiply and Screen
- **Soft Light**: Subtle contrast enhancement
- **Hard Light**: Strong contrast effect
- **Vivid Light**: Extreme contrast adjustments

### Selection Tools

#### Basic Selection Tools

- **Marquee tools**: Rectangular and elliptical selections
- **Lasso tools**: Freehand selection drawing
- **Magic Wand**: Color-based selection
- **Quick Selection**: Brush-based intelligent selection

#### Advanced Selection Tools

- **Object Selection**: AI-powered object detection
- **Select Subject**: Automatic main subject selection
- **Color Range**: Select by color similarity
- **Focus Area**: Select in-focus areas

#### Selection Refinement

- **Select and Mask**: Advanced edge refinement
- **Feather**: Soften selection edges
- **Expand/Contract**: Adjust selection size
- **Smooth**: Remove jagged selection edges

## Advanced Techniques

### Frequency Separation

#### Concept

- **Texture separation**: Separate skin texture from color/tone
- **High frequency**: Fine details and texture
- **Low frequency**: Color and tonal information
- **Independent editing**: Adjust texture and color separately

#### Process

1. **Duplicate layer twice**: Create high and low frequency layers
2. **Blur low frequency**: Remove texture with Gaussian blur
3. **Create high frequency**: Apply Image to subtract low from high
4. **Set blend mode**: Linear Light for high frequency layer
5. **Edit separately**: Retouch color and texture independently

### Luminosity Masks

#### Purpose

- **Precise tonal selections**: Select specific brightness ranges
- **Natural blending**: Selections follow natural light
- **Advanced color grading**: Targeted adjustments
- **Professional results**: Avoid selection artifacts

#### Creation Methods

- **Action-based**: Use pre-made actions for efficiency
- **Plugin tools**: TK Panel, Lumenzia, and similar tools
- **Blend If**: Alternative using layer style blend options
