### Log Categories

```bash
# Configure logging in named.conf
logging {
    channel default_log {
        file "/var/log/named/default.log" versions 3 size 5m;
        severity info;
        print-time yes;
        print-severity yes;
        print-category yes;
    };
    
    category default { default_log; };
    category queries { default_log; };
    category security { default_log; };
};
```

### Monitoring Tools

```bash
# Monitor DNS queries
dig @localhost example.com

# Check server status
rndc status

# View statistics
rndc stats
```
