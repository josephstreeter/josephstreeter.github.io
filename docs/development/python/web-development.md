## Overview

Python offers powerful frameworks for building web applications, from simple websites to complex enterprise applications.

## Popular Frameworks

### Django

Full-featured web framework with batteries included. See [Django Guide](web-development/django.md) for details.

### Flask

Lightweight and flexible micro-framework. See [Flask Guide](web-development/flask.md) for details.

### FastAPI

Modern, fast framework for building APIs. Ideal for microservices and async applications.
