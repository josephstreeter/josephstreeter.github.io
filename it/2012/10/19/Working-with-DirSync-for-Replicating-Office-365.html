Active Directory Account and Password Settings

Oct 19, 2012

Some helpful information about setting passwords and configuring accounts when programatically creating Active Directory user objects.

pwdLastSet Attribute

From MSDN:- Pwd-Last-Set attribute

The date and time that the password for this account was last changed. This value is stored as a large integer that represents the number of 100 nanosecond intervals since January 1, 1601 (UTC). If this value is set to 0 and the- User-Account-Control- attribute does not contain theUF_DONT_EXPIRE_PASSWD- flag, then the user must set the password at the next logon.
