What is Using That Network Port Anyways?

Oct 19, 2012

So, you see a list of ports that are open on your Windows host and you want to know what is using them. Well, you’re in luck. With a little command line-fu you can get the answer. “netstat -ano” will give you an output that includes the PID that is associated with each port number.

Then use the “tasklist” command to find your answer.
